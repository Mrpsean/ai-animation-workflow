Rules

  • ✓ Every scene must be understood in 2 seconds
  • ✓ Every scene takes place inside or on the body
  • ✓ Ask AI for 3+ image prompt options per scene — pick the best
  • ✓ When output is wrong, screenshot it → send to AI → get a fixed prompt
  • ✓ Animation prompts = one action, simply described
  • ✓ Keep all action centred in frame (safe zones)
  • ✗ No courtroom, no lineup, no politician — all tested, all flopped
  • ✗ Don't regenerate the same prompt hoping for different results
  • ✗ Don't use foods NOT on the test panel
  • ✗ Don't write long animation prompts — causes jitter

What you do

  1. Upload the character image and reference video breakdown to Claude or ChatGPT
  2. Give it the script. Ask for 3 visual options per scene. Pick the best.
  3. Paste the chosen prompt into Nano-Banana Pro (2K). Generate 2-3 versions.
  4. If the output is wrong: screenshot → AI → fixed prompt → regenerate
  5. Once all stills are done: send them for review before animating
  6. Animate each still in Seedance / Kling. One simple action per clip.
  7. Add voiceover. If it sounds off, run through ElevenLabs Voice Changer.
  8. Stitch, caption (white text, black outline), export 9:16 + 4:5.
Checkpoint 1: Send all stills for review BEFORE animating. Regenerating a still is free. Redoing animation wastes credits.
Checkpoint 2: Send final video for review before delivery.
The core skill: When something doesn't look right, don't just regenerate and hope. Screenshot the problem, describe what's wrong, send it to Claude/ChatGPT, and get a fixed prompt. That feedback loop is the whole game.
